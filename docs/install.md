1.发布方式：使用 Nuitka 直接编译打包为独立安装包，支持一键安装、直接部署。

2.运行环境：打包后程序不依赖 Python 环境、不依赖第三方库、无需配置运行依赖，解压 / 安装即可独立运行。

3.依赖库：开发阶段使用 PySide6、ONNX Runtime、Simple ITK、VTK 等库构建，所有依赖已内置打包到安装包中，用户无额外安装步骤。
